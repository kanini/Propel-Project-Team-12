# User Story - US_003

## Story ID
   * ID Format: US_003

## Story Title
   * PostgreSQL Database Setup with pgvector Extension

## Description
  * As a developer, I want to provision a PostgreSQL 16 database with pgvector extension on Supabase, so that I can store relational healthcare data with referential integrity and support AI vector embeddings for semantic search in medical code mapping.

## Acceptance Criteria
  * **Given** I have a Supabase account, **When** I create a new project, **Then** a PostgreSQL 16 database is provisioned with connection string, credentials, and SSL certificate available.

  * **Given** the database is provisioned, **When** I execute the pgvector extension installation script, **Then** the extension is enabled and I can create tables with vector(1536) columns for AI embeddings.

  * **Given** Entity Framework Core is configured, **When** I run the connection test, **Then** the application successfully connects to Supabase PostgreSQL using the connection string with SSL mode required.

  * **Given** the database context is configured, **When** I run `dotnet ef migrations add InitialCreate`, **Then** a migration file is generated with all entity configurations from design.md domain model.

  * **Given** migrations are created, **When** I run `dotnet ef database update`, **Then** all tables, indexes, constraints, and the pgvector extension are applied successfully to the Supabase database.

  * **Given** backup configuration is defined, **When** I check Supabase settings, **Then** point-in-time recovery is enabled with 15-minute granularity as required by DR-008.

## Edge Cases
   * What happens when Supabase free tier storage limit is exceeded? (Monitor usage and display warning at 80% threshold with upgrade path)
   * How does the system handle SSL certificate validation failures? (Provide clear error message with troubleshooting steps for SSL configuration)
   * What if migration fails midway? (Use transactional migrations to rollback automatically; log detailed error for manual intervention)
