# User Story - US_027

## Story ID

* ID: US_027

## Story Title

* Cancel and Reschedule Appointments

## Description

* As a patient, I want to cancel or reschedule my appointments with configurable advance notice requirements, so that I can manage my schedule while respecting the provider's booking policies.

## Acceptance Criteria

1. **Given** I have an upcoming appointment, **When** I click "Cancel" from My Appointments and confirm, **Then** the appointment status changes to Cancelled, the slot is released for other patients, a cancellation confirmation is sent, and the calendar event is removed.

2. **Given** I want to reschedule, **When** I click "Reschedule", **Then** the system displays available alternative slots for the same provider and I can select a new date/time.

3. **Given** I select a new slot for rescheduling, **When** I confirm, **Then** the original slot is released, the new appointment is created, a reschedule confirmation is sent, and the calendar event is updated.

4. **Given** cancellation notice is configured (e.g., 24 hours), **When** I attempt to cancel within the restricted window, **Then** the system displays the cancellation policy and may prevent cancellation based on configuration.

## Edge Cases

* What happens when the rescheduled slot becomes unavailable during the process? System displays conflict message and refreshes available slots.
* How does the system handle cancelling a waitlisted appointment? Waitlist entry is removed and the patient is notified.
