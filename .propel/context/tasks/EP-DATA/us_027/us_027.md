# User Story - US_027

## Story ID
* ID: US_027

## Story Title
* Data Integrity Unit Tests

## Description
* As a QA engineer, I want comprehensive xUnit tests validating unique constraints, foreign key cascades, optimistic concurrency conflicts, and audit log immutability, so that database integrity is automatically verified in CI/CD pipelines before deployment.

## Acceptance Criteria
1. **Given** xUnit test project exists (US_006), **When** I create data integrity test suite, **Then** it includes test categories:
   - **Unique Constraints**: Email uniqueness for Patient/Staff/Admin, appointment slot uniqueness per provider
   - **Foreign Key Cascades**: Deleting patient cascades to appointments/documents (or restricts based on design decision)
   - **Optimistic Concurrency**: Concurrent appointment bookings trigger `DbUpdateConcurrencyException`
   - **Audit Log Immutability**: UPDATE/DELETE attempts on AuditLogs throw exceptions

2. **Given** unique constraint tests are written, **When** I run them, **Then** I verify:
   - Insert patient with duplicate email throws `DbUpdateException` with constraint violation
   - Insert appointment with duplicate slot (same ProviderID + SlotDateTime) throws exception
   - Test passes if exception message contains expected constraint name (e.g., "IX_Patients_Email_Unique")

3. **Given** foreign key cascade tests exist, **When** I execute them, **Then** I confirm:
   - Deleting Patient with appointments either cascades (deletes appointments) or restricts (throws exception) based on FK configuration
   - Deleting ClinicalDocument cascades to ExtractedData (orphan extraction records prevented)
   - Test validates expected row counts before and after delete operation

4. **Given** all tests pass, **When** CI/CD pipeline runs, **Then** data integrity test suite:
   - Executes against Testcontainers PostgreSQL instance (isolated from dev/prod)
   - Reports pass/fail status to GitHub Actions workflow
   - Blocks merge if any test fails

## Edge Cases
* What happens when tests run concurrently and conflict with each other? (Each test uses isolated Testcontainers instance or transaction rollback to avoid conflicts)
* How does system handle slow-running tests (>30 seconds)? (Mark as `[Fact(Timeout = 30000)]` to fail if exceeds timeout; optimize test using in-memory database for non-integration scenarios)

- **Unique constraint test example:**
  ```csharp
  public class UniqueConstraintTests : IAsyncLifetime
  {
      private PostgreSqlContainer _postgresContainer;
      private ApplicationDbContext _context;

      public async Task InitializeAsync()
      {
          _postgresContainer = new PostgreSqlBuilder()
              .WithImage("pgvector/pgvector:pg16")
              .Build();

          await _postgresContainer.StartAsync();

          var options = new DbContextOptionsBuilder<ApplicationDbContext>()
              .UseNpgsql(_postgresContainer.GetConnectionString())
              .Options;

          _context = new ApplicationDbContext(options);
          await _context.Database.MigrateAsync();
      }

      [Fact]
      public async Task Patient_DuplicateEmail_ThrowsDbUpdateException()
      {
          // Arrange
          var patient1 = new Patient { PatientID = Guid.NewGuid(), Email = "duplicate@example.com", FirstName = "John", LastName = "Doe", DateOfBirth = DateTime.Parse("1990-01-01") };
          var patient2 = new Patient { PatientID = Guid.NewGuid(), Email = "duplicate@example.com", FirstName = "Jane", LastName = "Smith", DateOfBirth = DateTime.Parse("1992-05-15") };

          _context.Patients.Add(patient1);
          await _context.SaveChangesAsync();

          // Act & Assert
          _context.Patients.Add(patient2);
          var exception = await Assert.ThrowsAsync<DbUpdateException>(() => _context.SaveChangesAsync());

          exception.Message.Should().Contain("IX_Patients_Email_Unique");
      }

      [Fact]
      public async Task Appointment_DuplicateSlot_ThrowsDbUpdateException()
      {
          // Arrange
          var providerId = Guid.NewGuid();
          var slotDateTime = DateTime.UtcNow.AddDays(1);

          var appointment1 = new Appointment { AppointmentID = Guid.NewGuid(), PatientID = Guid.NewGuid(), ProviderID = providerId, SlotDateTime = slotDateTime, Status = AppointmentStatus.Scheduled };
          var appointment2 = new Appointment { AppointmentID = Guid.NewGuid(), PatientID = Guid.NewGuid(), ProviderID = providerId, SlotDateTime = slotDateTime, Status = AppointmentStatus.Scheduled };

          _context.Appointments.Add(appointment1);
          await _context.SaveChangesAsync();

          // Act & Assert
          _context.Appointments.Add(appointment2);
          var exception = await Assert.ThrowsAsync<DbUpdateException>(() => _context.SaveChangesAsync());

          exception.Message.Should().Contain("IX_Appointments_SlotDateTime_ProviderID_Unique");
      }

      public async Task DisposeAsync()
      {
          await _context.DisposeAsync();
          await _postgresContainer.DisposeAsync();
      }
  }
  ```

- **Optimistic concurrency test:**
  ```csharp
  [Fact]
  public async Task Appointment_ConcurrentBooking_ThrowsConcurrencyException()
  {
      // Arrange: Create appointment
      var appointment = new Appointment { AppointmentID = Guid.NewGuid(), PatientID = Guid.NewGuid(), ProviderID = Guid.NewGuid(), SlotDateTime = DateTime.UtcNow.AddDays(1), Status = AppointmentStatus.Scheduled };
      _context.Appointments.Add(appointment);
      await _context.SaveChangesAsync();

      // Act: Simulate concurrent modification
      var context2 = new ApplicationDbContext(_context.Database.GetConnectionString());
      var appointment2 = await context2.Appointments.FindAsync(appointment.AppointmentID);
      appointment2.Status = AppointmentStatus.Arrived;

      // First update succeeds
      appointment.Status = AppointmentStatus.Cancelled;
      await _context.SaveChangesAsync();

      // Second update throws concurrency exception
      var exception = await Assert.ThrowsAsync<DbUpdateConcurrencyException>(() => context2.SaveChangesAsync());
      exception.Should().NotBeNull();
  }
  ```

- **Audit log immutability test:**
  ```csharp
  [Fact]
  public async Task AuditLog_UpdateAttempt_ThrowsException()
  {
      // Arrange
      var auditLog = new AuditLog { AuditLogID = Guid.NewGuid(), UserID = Guid.NewGuid(), UserRole = UserRole.Patient, Action = AuditAction.Create, EntityType = "Patient", EntityID = Guid.NewGuid(), Timestamp = DateTime.UtcNow, IPAddress = "192.168.1.1", UserAgent = "Test" };
      _context.AuditLogs.Add(auditLog);
      await _context.SaveChangesAsync();

      // Act & Assert: Attempt UPDATE
      auditLog.Action = AuditAction.Update;
      var exception = await Assert.ThrowsAsync<DbUpdateException>(() => _context.SaveChangesAsync());

      exception.InnerException.Message.Should().Contain("Audit logs are immutable");
  }

  [Fact]
  public async Task AuditLog_DeleteAttempt_ThrowsException()
  {
      // Arrange
      var auditLog = new AuditLog { AuditLogID = Guid.NewGuid(), UserID = Guid.NewGuid(), UserRole = UserRole.Staff, Action = AuditAction.Read, EntityType = "Appointment", EntityID = Guid.NewGuid(), Timestamp = DateTime.UtcNow, IPAddress = "10.0.0.1", UserAgent = "Test" };
      _context.AuditLogs.Add(auditLog);
      await _context.SaveChangesAsync();

      // Act & Assert: Attempt DELETE
      _context.AuditLogs.Remove(auditLog);
      var exception = await Assert.ThrowsAsync<DbUpdateException>(() => _context.SaveChangesAsync());

      exception.InnerException.Message.Should().Contain("Audit logs are immutable");
  }
  ```

- **Foreign key cascade test:**
  ```csharp
  [Fact]
  public async Task ClinicalDocument_DeleteCascadesToExtractedData()
  {
      // Arrange
      var document = new ClinicalDocument { DocumentID = Guid.NewGuid(), PatientID = Guid.NewGuid(), DocumentType = DocumentType.Lab, UploadedAt = DateTime.UtcNow, FilePath = "/path/to/doc.pdf", FileSize = 1024, ProcessingStatus = ProcessingStatus.Completed };
      var extractedData = new ExtractedData { ExtractedDataID = Guid.NewGuid(), DocumentID = document.DocumentID, DataType = DataType.Vital, Value = "{\"bp\": \"120/80\"}", ConfidenceScore = 0.9m, ExtractedAt = DateTime.UtcNow };

      _context.ClinicalDocuments.Add(document);
      _context.ExtractedData.Add(extractedData);
      await _context.SaveChangesAsync();

      var extractedDataCountBefore = await _context.ExtractedData.CountAsync(e => e.DocumentID == document.DocumentID);
      extractedDataCountBefore.Should().Be(1);

      // Act: Delete document
      _context.ClinicalDocuments.Remove(document);
      await _context.SaveChangesAsync();

      // Assert: Verify cascade delete
      var extractedDataCountAfter = await _context.ExtractedData.CountAsync(e => e.DocumentID == document.DocumentID);
      extractedDataCountAfter.Should().Be(0);
  }
  ```
