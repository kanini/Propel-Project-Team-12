# User Story - US_026

## Story ID
* ID: US_026

## Story Title
* Repository Pattern Implementations

## Description
* As a backend developer, I want repository pattern interfaces and Entity Framework implementations for all domain entities, so that data access logic is abstracted from business services and can be unit tested with mocked repositories.

## Acceptance Criteria
1. **Given** the repository pattern is adopted, **When** I define interfaces, **Then** I have:
   - `IPatientRepository` with methods: GetByIdAsync, GetByEmailAsync, AddAsync, UpdateAsync, DeleteAsync (soft delete)
   - `IAppointmentRepository` with methods: GetByIdAsync, GetByPatientIdAsync, GetAvailableSlotsAsync, BookAppointmentAsync (handles optimistic concurrency)
   - `IClinicalDocumentRepository` with methods: GetByPatientIdAsync, AddAsync, SearchByEmbeddingAsync (pgvector similarity search)
   - `IExtractedDataRepository` with methods: GetLowConfidenceExtractions, AddAsync, VerifyByStaffAsync
   - `IAuditLogRepository` with methods: AddAsync, GetByUserIdAsync, GetByEntityAsync (read-only, no update/delete)

2. **Given** EF Core implementations exist, **When** I review the code, **Then** each repository:
   - Inherits from base `Repository<T>` class with common CRUD operations
   - Implements specialized queries in derived repository (e.g., `AppointmentRepository.GetAvailableSlotsAsync`)
   - Uses `IQueryable<T>` for deferred execution and query optimization
   - Handles exceptions with descriptive error messages (e.g., `NotFoundException`, `ConcurrencyException`)

3. **Given** Unit of Work pattern is implemented, **When** I inspect `IUnitOfWork`, **Then** it:
   - Exposes all repository interfaces as properties: `IPatientRepository Patients { get; }`, `IAppointmentRepository Appointments { get; }`, etc.
   - Provides `SaveChangesAsync()` method to commit all repository changes in single transaction
   - Implements `IDisposable` for DbContext lifecycle management

4. **Given** repositories are registered in DI container, **When** I review `Program.cs`, **Then** I see:
   - `services.AddScoped<IUnitOfWork, UnitOfWork>();`
   - `services.AddScoped<IPatientRepository, PatientRepository>();`
   - All repository interfaces registered with scoped lifetime (matches DbContext)

## Edge Cases
* What happens when repository method is called without active DbContext? (Throws `InvalidOperationException: DbContext disposed`; document proper DI scope usage)
* How does system handle repository query timeouts on slow queries? (Configure command timeout in DbContext options; log slow queries for optimization)

## Technical Notes
- Base repository interface and implementation:
  ```csharp
  public interface IRepository<T> where T : class
  {
      Task<T?> GetByIdAsync(Guid id);
      Task<IEnumerable<T>> GetAllAsync();
      Task AddAsync(T entity);
      void Update(T entity);
      void Delete(T entity);
  }

  public class Repository<T> : IRepository<T> where T : class
  {
      protected readonly ApplicationDbContext _context;
      protected readonly DbSet<T> _dbSet;

      public Repository(ApplicationDbContext context)
      {
          _context = context;
          _dbSet = context.Set<T>();
      }

      public virtual async Task<T?> GetByIdAsync(Guid id)
      {
          return await _dbSet.FindAsync(id);
      }

      public virtual async Task<IEnumerable<T>> GetAllAsync()
      {
          return await _dbSet.ToListAsync();
      }

      public virtual async Task AddAsync(T entity)
      {
          await _dbSet.AddAsync(entity);
      }

      public virtual void Update(T entity)
      {
          _dbSet.Update(entity);
      }

      public virtual void Delete(T entity)
      {
          _dbSet.Remove(entity);
      }
  }
  ```

- Specialized repository example:
  ```csharp
  public interface IAppointmentRepository : IRepository<Appointment>
  {
      Task<IEnumerable<Appointment>> GetByPatientIdAsync(Guid patientId);
      Task<IEnumerable<AppointmentSlot>> GetAvailableSlotsAsync(Guid providerId, DateTime date);
      Task<Appointment> BookAppointmentAsync(Appointment appointment); // Handles concurrency
  }

  public class AppointmentRepository : Repository<Appointment>, IAppointmentRepository
  {
      public AppointmentRepository(ApplicationDbContext context) : base(context) { }

      public async Task<IEnumerable<Appointment>> GetByPatientIdAsync(Guid patientId)
      {
          return await _dbSet
              .Where(a => a.PatientID == patientId)
              .OrderByDescending(a => a.SlotDateTime)
              .ToListAsync();
      }

      public async Task<IEnumerable<AppointmentSlot>> GetAvailableSlotsAsync(Guid providerId, DateTime date)
      {
          return await _context.AppointmentSlots
              .Where(s => s.ProviderID == providerId && 
                         s.StartDateTime.Date == date.Date &&
                         s.IsAvailable == true)
              .OrderBy(s => s.StartDateTime)
              .ToListAsync();
      }

      public async Task<Appointment> BookAppointmentAsync(Appointment appointment)
      {
          try
          {
              await AddAsync(appointment);
              await _context.SaveChangesAsync();
              return appointment;
          }
          catch (DbUpdateConcurrencyException ex)
          {
              throw new ConcurrencyException("Slot no longer available", ex);
          }
      }
  }
  ```

- Unit of Work pattern:
  ```csharp
  public interface IUnitOfWork : IDisposable
  {
      IPatientRepository Patients { get; }
      IAppointmentRepository Appointments { get; }
      IClinicalDocumentRepository ClinicalDocuments { get; }
      IExtractedDataRepository ExtractedData { get; }
      IAuditLogRepository AuditLogs { get; }

      Task<int> SaveChangesAsync();
  }

  public class UnitOfWork : IUnitOfWork
  {
      private readonly ApplicationDbContext _context;

      public IPatientRepository Patients { get; }
      public IAppointmentRepository Appointments { get; }
      public IClinicalDocumentRepository ClinicalDocuments { get; }
      public IExtractedDataRepository ExtractedData { get; }
      public IAuditLogRepository AuditLogs { get; }

      public UnitOfWork(ApplicationDbContext context)
      {
          _context = context;
          Patients = new PatientRepository(context);
          Appointments = new AppointmentRepository(context);
          ClinicalDocuments = new ClinicalDocumentRepository(context);
          ExtractedData = new ExtractedDataRepository(context);
          AuditLogs = new AuditLogRepository(context);
      }

      public async Task<int> SaveChangesAsync()
      {
          return await _context.SaveChangesAsync();
      }

      public void Dispose()
      {
          _context.Dispose();
      }
  }
  ```

- Service layer usage:
  ```csharp
  public class BookingService
  {
      private readonly IUnitOfWork _unitOfWork;

      public BookingService(IUnitOfWork unitOfWork)
      {
          _unitOfWork = unitOfWork;
      }

      public async Task<Appointment> BookAppointmentAsync(Guid patientId, Guid providerId, DateTime slotDateTime)
      {
          var patient = await _unitOfWork.Patients.GetByIdAsync(patientId);
          if (patient == null) throw new NotFoundException("Patient not found");

          var appointment = new Appointment
          {
              PatientID = patientId,
              ProviderID = providerId,
              SlotDateTime = slotDateTime,
              Status = AppointmentStatus.Scheduled
          };

          await _unitOfWork.Appointments.BookAppointmentAsync(appointment);
          await _unitOfWork.SaveChangesAsync();

          return appointment;
      }
  }
  ```

- Unit testing with mocked repositories:
  ```csharp
  [Fact]
  public async Task BookAppointmentAsync_ValidData_ReturnsAppointment()
  {
      // Arrange
      var mockUnitOfWork = new Mock<IUnitOfWork>();
      mockUnitOfWork.Setup(uow => uow.Patients.GetByIdAsync(It.IsAny<Guid>()))
          .ReturnsAsync(new Patient { PatientID = Guid.NewGuid() });
      mockUnitOfWork.Setup(uow => uow.Appointments.BookAppointmentAsync(It.IsAny<Appointment>()))
          .ReturnsAsync((Appointment a) => a);

      var service = new BookingService(mockUnitOfWork.Object);

      // Act
      var result = await service.BookAppointmentAsync(Guid.NewGuid(), Guid.NewGuid(), DateTime.UtcNow);

      // Assert
      Assert.NotNull(result);
      mockUnitOfWork.Verify(uow => uow.SaveChangesAsync(), Times.Once);
  }
  ```
