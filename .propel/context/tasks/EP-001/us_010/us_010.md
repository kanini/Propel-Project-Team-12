# User Story - US_010

## Story ID
* ID: US_010

## Story Title
* View Appointment History

## Description
* As a patient, I want to view my complete appointment history with past and upcoming appointments, so that I can track my healthcare visits and manage future appointments.

## Acceptance Criteria
1. **Given** I am logged into my patient dashboard, **When** I navigate to the "Appointment History" section, **Then** the system displays a list of all my appointments (past and upcoming) sorted by date descending, with status badges (Scheduled/Arrived/Completed/Cancelled/NoShow).

2. **Given** I am viewing my appointment history, **When** an appointment is in "Scheduled" status and the appointment date is in the future, **Then** I see "Reschedule" and "Cancel" action buttons for that appointment.

3. **Given** I click the "Reschedule" button for a scheduled appointment, **When** I select a new appointment slot, **Then** the system updates the appointment time, sends confirmation email with updated details, updates my calendar, and displays "Appointment rescheduled successfully."

4. **Given** I click the "Cancel" button for a scheduled appointment, **When** I confirm cancellation, **Then** the system cancels the appointment, updates status to "Cancelled," releases the slot to availability, sends cancellation confirmation email, and removes the event from my linked calendar.

## Edge Cases
* What happens when I try to reschedule an appointment that is within 24 hours? (System displays warning "Appointments within 24 hours cannot be rescheduled online. Please call [phone number] for assistance" and disables Reschedule button)
* How does the system handle canceled appointments with pending preferred slot swap requests? (System cancels swap request, updates swap status to "Cancelled," releases primary slot, notifies system that patient no longer interested in swap)
* What happens if I have a very long appointment history (>100 appointments)? (System implements pagination with 20 appointments per page, provides filter controls for date range and status, displays appointment count summary at top)
* How does the system display appointments that were rescheduled multiple times? (Each appointment shows current date/time with "Rescheduled" badge; expandable details reveal original booking date and reschedule history with timestamps)
