# User Story - US_037

## Story ID

- ID: US_037

## Story Title

- Appointment Selection for Intake

## Description

- As a patient with multiple upcoming appointments, I want to select which appointment I want to complete intake for, so that I can fill out the appropriate intake form for my specific visit.

## Acceptance Criteria

1. **Given** I have multiple upcoming appointments requiring intake, **When** I navigate to the intake page or click the intake button, **Then** I see a list of all my upcoming appointments with provider name, date, time, and intake status (pending/completed/in-progress).

2. **Given** I see the appointment selection list, **When** I select a specific appointment, **Then** I am taken to the intake mode selection screen (AI or Manual) for that appointment, and the intake session is initialized with the selected appointment ID.

3. **Given** I have completed intake for some appointments, **When** I view the appointment list, **Then** completed appointments display a green "Completed" badge and an "Edit Intake" action button, while pending appointments show a "Start Intake" button.

4. **Given** I have an in-progress intake session, **When** I view the appointment list, **Then** the appointment with in-progress intake displays an "In Progress" badge and a "Continue Intake" button that resumes the session.

5. **Given** I have only one upcoming appointment, **When** I click the intake button, **Then** I am taken directly to the intake mode selection screen for that appointment, bypassing the appointment list.

6. **Given** I have no upcoming appointments requiring intake, **When** I access the intake page, **Then** I see an empty state message "No upcoming appointments requiring intake" with a "Book Appointment" link.

## Edge Cases

- What happens when an appointment date has passed but intake was not completed? Appointment is removed from the intake-required list and marked as "Past due" in the appointment history.
- How does the system handle appointments that don't require intake (e.g., follow-ups with no new info needed)? Only appointments flagged as requiring intake are shown in the selection list.
- What if the patient starts intake for one appointment but wants to switch to another? The current intake session is auto-saved, and starting intake for a different appointment creates a new session without losing previous work.
