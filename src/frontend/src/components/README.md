# Components

This directory contains reusable React components.

## Structure

- **common/**: Shared UI components (buttons, inputs, cards, etc.)
- **layout/**: Layout components (headers, footers, navigation, etc.)
- **forms/**: Form components and form-related utilities

## Guidelines

- Each component should be in its own directory with an index.ts file
- Include PropTypes or TypeScript interfaces for all components
- Write unit tests for components with complex logic
- Use Tailwind CSS for styling
